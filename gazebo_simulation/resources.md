# Resources and Citations

## Official Documentation & Tutorials
- [ROS Wiki](https://wiki.ros.org)  
  Comprehensive documentation for ROS 1, including tutorials, API references, and packages.

- [Gazebo Tutorials](http://gazebosim.org/tutorials)  
  Official tutorials for getting started with Gazebo simulation.

- [ROS Wiki: Gazebo](https://wiki.ros.org/gazebo)  
  Integration details and usage of Gazebo simulator with ROS.

## Repositories
- [ROS GitHub Repository](https://github.com/ros/ros)  
  The official ROS source code and issue tracker.

- [Gazebo GitHub Repository](https://github.com/osrf/gazebo)  
  Source code for Gazebo simulator.

## Books and Papers
- Quigley, M., Conley, K., Gerkey, B., Faust, J., Foote, T., Leibs, J., Wheeler, R., & Ng, A. Y. (2009). ROS: An open-source Robot Operating System. In *ICRA Workshop on Open Source Software* (Vol. 3, No. 3.2, p. 5). Kauai, HI. Retrieved from http://www.willowgarage.com/sites/default/files/icraoss09-ROS.pdf

- Koenig, N., & Howard, A. (2004). Design and use paradigms for Gazebo, an open-source multi-robot simulator. In *Proceedings of the IEEE/RSJ International Conference on Intelligent Robots and Systems* (pp. 2149–2154). IEEE. https://doi.org/10.1109/IROS.2004.1389727

## Additional Learning
- [ROS Tutorials - The Construct](https://www.theconstructsim.com/ros-tutorials/)  
  Interactive ROS tutorials and courses.

- [ROS Answers](https://answers.ros.org/questions/)  
  Community Q&A for ROS users.
